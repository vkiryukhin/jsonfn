# JSONfn

Javascript (both node.js and browser) plugin to stringify / parse / clone jsvascript objects with

  - **Functions** 
  - **RegExp** 
  - **Date**


**Version** - 0.60.00

**License:** MIT

**Basic usage:**

    var str = JSONfn.stringify(obj);

    var obj = JSONfn.parse(str);
